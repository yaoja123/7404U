## Design Decisions

### Canvas & Grid
- A0 portrait: 33.11" × 46.81"
- **12-column grid**: `MARGIN = 1.1"`, `GUTTER = 0.30"`, column width ≈ 2.36"
- **Vertical rhythm**: `SECTION_GAP = 0.90"`, `CARD_PAD = 0.45"`
- Every shape's x/width snaps to grid columns — no freehand positioning

### Typography Scale
Strict 7-level scale to fix v3's irregular jumps (21 → 62pt with nothing between):

| Constant | Size | Usage |
|---|---|---|
| `T_TITLE` | 90pt | Poster title |
| `T_H1` | 52pt | Section header |
| `T_H2` | 36pt | Card title |
| `T_METRIC` | 88pt | Big result numbers |
| `T_BODY` | 26pt | Body text |
| `T_SMALL` | 21pt | Secondary body |
| `T_CAPTION` | 17pt | Captions / footnotes |

Fonts: **Aptos Display** (headings), **Aptos** (body), **Courier New** (formulas)

### Colour Palette (minimalist, vs v3's pastels)

| Name | Hex | Role |
|---|---|---|
| `bg` | `#F7F6F2` | Warm off-white ground |
| `ink` | `#0E1116` | Near-black text |
| `muted` | `#5B6470` | Secondary text |
| `accent` | `#C1121F` | Crimson — MaxSup / wrong prediction |
| `navy` | `#1E3A8A` | Deep navy — LS / comparison side |
| `green` | `#1B7A4E` | Positive result indicators |
| `hairline` | `#E0DDD7` | Section dividers |

**No pastels. No background ovals.** Depth comes from whitespace + hairline rules only.

### Hero Design Rationale
The hero is the piece v3 failed most on (generic text, no visual hierarchy).

- **Left panel**: shows *why* LS fails — fox logit (2.8) above cat (1.0), but LS still points its arrow at cat. Fox row highlighted in crimson so it's readable at 3 m.
- **Right panel**: side-by-side bar charts, one navy (LS arrow on cat) one crimson (MaxSup arrow on fox). The visual difference is immediate.
- **Caption**: single sentence that states the thesis.

### What Was Dropped from v3
- Background pastel ovals (decorative, not semantic)
- `add_cluster_icon` function (procedural dots didn't read clearly)
- `add_strip` function (dead code, never used)
- Supporting Evidence section (too dense; poster_tips: "people will ignore most things")
- Drop shadow on cards (adds noise in minimalist style)
