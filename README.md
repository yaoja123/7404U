`MaxSup: Overcoming Representation Collapse in Label Smoothing`

## Working Rules

- Edit `poster/MaxSup_A0_HiFi_Poster_v3.pptx` only if you are making the main draft better.
- If you want to try a different design direction, duplicate it as a new version first.
- Do not overwrite archived files.
- Keep text short and presentation-oriented. This is a poster, not a paper.
- Put `Group ID` and all teammate names on the final poster.

## Suggested Workflow

1. Read `TODO.md`
2. Check `TEAM.md`
3. Open the latest PPT in `poster/`
4. If making a major change, create a new version
5. Export a preview image after major visual changes

## Team Coordination

This repo is meant to reduce confusion, not add pressure.

- `TEAM.md` is only a light suggestion for 3 people
- `TODO.md` is a reminder list, not a strict checklist
- the main thing is to keep everyone editing the correct latest file

## Submission Reminder

Final deliverable should be:

- `A0 portrait`
- `editable PPT`
- visually readable from distance
- concise, not paper-dense
